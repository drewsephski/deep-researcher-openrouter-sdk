# Deep Research AI Agent

A powerful AI research agent that generates follow-up questions, crafts optimal search queries, and compiles comprehensive research reports using multiple LLMs and web search capabilities.

## Technologies Used

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Styling:** Tailwind CSS v4, Shadcn UI
- **AI Integration:** Vercel AI SDK v6
- **LLMs:** GPT, Claude, Gemini, Deepseek, etc. via OpenRouter
- **Web Search:** Exa Search API
- **UI Components:** Shadcn
- **Language:** TypeScript
- **Package Manager:** Bun
